What is a DOMAIN and what is a RANGE?

Domain is the set of all x-values while the Range is the set of all y-values. Note that we can only write the same element in the domain or range once.

However, in Domain and Range of a Function, Domain is the set of all values of x that give real values for y. Similarly, Range is the set of all values of y { now as f(x)} that gives the real values for x

Lets talk about the three types of correspondences:

One-to-One Correspondence
- refers to where one element of the domain is mapped to one element of the range.

Many-to-One Correspondence
- refers to where two or more elements of the domain are mapped to the same element of the range.

One-to-Many Correspondence
- refers to where one element of the domain is mapped to two or more elements of the range.
