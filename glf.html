What is a Linear Function?

A Linear Function is defined by
f(x) = mx + b
where m and b are real numbers. Its graph is a line with slope (m) and y-intercept (b).
The rule of what is a function also applies here as not all Linear Equations are Linear Functions, basically its a linear function, if its a straight line and if the x-value is only matching to one y-value.

See the Difference between the forms:

(Slope-Intercept Form)
y = mx + b
(Linear Function)
f(x) = mx + b

Remember:

❑ If m ≠ 0, the the degree ofthe function is 1. The line here is diagonal.
❑ If m = 0 and b ≠ 0, then the degree of the function is 0. The line here is horizontal.
❑ If m = 0 and b = 0, then the degree of the function is undefined. The line does not exist.

Examples:

Using Slope and Y-Intercept:
f(x) = - 4/3x + 2
m = - 4/3
y-intercept = (0,2)

Using Slope and Y-Intercept:
f(x) = 5
m = 0
y-intercept = (0,5)

Using Table of Values:
f(x) = 5x + 4
Table of Values:
x -2 1 3
f(x) -6 9 19
