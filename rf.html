What is a RELATION and what is a FUNCTION?

A relation is a relationship between sets of values of ordered pairs. While a function is a relation in which each value from the set of ordered pairs' first components is associated with exactly one value from the set of ordered pairs' second components.

There are three representations of relations:

through (1) table, (2) mapping diagram, (3) graph, and (4) rule. but for this website, we'll be showing you examples using Set, Mapping Diagram, and Table of Values

Remember: We cam easily determine if a relation is a function by simply looking if the first element is used only once in whatever given set. Remember the following to identify a function:

  1. Each element in x must be matched by an element in y.
  2. Some elements in y can be not matched with an element in x.
  3. Two to more elements of x can be matched with the same element in y.
All functions are relations but some relations are not functions.
