Before we start, what is a RELATION and what is a FUNCTION?

A relation is a relationship between sets of values of ordered pairs. While a function is a relation in which each value from the set of ordered pairs' first components is associated with exactly one value from the set of ordered pairs' second components.

There are three representations of relations:

through (1) table, (2) mapping diagram, (3) graph, and (4) rule. but for this website, we'll be showing you examples using Set, Mapping Diagram, and Table of Values
